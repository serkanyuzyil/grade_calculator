# Weighted Grade Calculator

## Overview
This is a Python-based utility tool designed to calculate the final academic grade based on user-defined assessment components. This program allows the user to input custom specific weights for Midterms, Finals, and Homework assignments, providing a flexible solution for different course grading schemas.

## Key Features
* **Customizable Weighting:** Users can define specific percentage weights for each assessment (e.g., 30% Midterm, 50% Final, 20% Homework).
* **Dynamic Input:** The script interacts with the user to accept various score inputs.
* **Immediate Calculation:** Instantly computes the weighted average based on the provided data.

## How to Run
1.  Run the script using a Python interpreter (terminal or IDE).
2.  Follow the on-screen prompts to enter your **raw scores** for each category.
3.  Enter the corresponding **percentage weight** for each category when prompted.
4.  The program will display your calculated final grade.

## Technologies Used
* Python 3.13.7
